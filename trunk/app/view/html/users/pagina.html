O arquivo da foto deve ter o tamanho de 300px X 400px (ou outra resolução com relação 3 X 4). O tamanho máximo do arquivo não pode superar 1Mb.

Se não deseja trocar a senha deixe o campo "Nova Senha" em branco.




Digite palavras no campo "Tags" (separadas por vírgula) indicando sua área de trabalho, preferências, interesses (etc), para facilitar a busca.

Por exemplo: "php,neos,design,javascript".

Para o campo "Contatos", faça uma listagem com seus dados de contato.
Ex.:
Skype: neosphp
Msn: neosphp@hotmail.com
